PLEASE READ BEFORE ANALYZYING/BUILDING


This project was built upon wompwomp6's base (https://github.com/wompwomp6/cs2-skin-changer) and "vibe coded" using Gemini 3 Pro.

Build Instructions

Please refer to **BUILDING.md** for detailed, step-by-step build instructions.
We have also included a `setup_project.bat` script that will automatically download NuGet and restore the packages for you.

To build:
1.  Run `setup_project.bat`.
2.  Open `src/ext-cs2-skin-changer.sln` in Visual Studio 2022.
3.  Build (Release/x64).

Features
-   Skin Changer
-   Config Manager
-   Auto Update Offsets

Note
Release builds are included in the `bin` folder for your convenience, along with PDBs.
